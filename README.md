# Personal website

Personal website built with Jekyll and styled from the `charlieoneill11.github.io` source.

## Edit your content

- Change the site name in `_config.yml`.
- Replace the homepage bio and email in `index.html`.
- Replace `images/profile-placeholder.svg` with your own profile photo and update the image path in `index.html`.
- Add research or project entries in `research.html`.
- Add posts as Markdown files under `_posts/` using names like `2026-05-08-my-post.md`.

## Local Development

```bash
bundle install
bundle exec jekyll serve --livereload
```

Site will be at http://127.0.0.1:4000.
